『ぼくのかっこわるくて、いじわるで、大嫌いなおじ』で第85回アカデミー賞脚本賞、前作『ガールズ・オブ・サマー』で第76回金獅子賞(最高賞)を獲得し、いまや名実ともに世界を代表する映画監督のひとりとなったエミリー・ブラウン。
元はアクション女優として活躍していた彼女は、スタント事故をきっかけに2005年より監督業に専念。
クライムサスペンス、スペースオペラ、ラブロマンスと立て続けに複数のジャンル映画を製作し、常に新しいことに挑戦してきた。

最新作では一転、唯一無二の永遠の美を追い求めるモデル業界での渦巻く欲望と夢見る少女たちがいかにして無慈悲かつ残酷に変貌していくのかを幻想的かつ煌びやかな映像で描き出すサイコスリラーに挑戦。

映画スタジオは『アンダー・ザ・スカイ』が第90回アカデミー賞で計5部門を受賞したことで話題のB42製作。
今作ではエミリー・ブラウン初のダブル主演映画となる。
トップモデルを夢見る少女アン役には、『ラブ&ロック』シリーズのリンダ役で絶大な人気を博した、才能あふれる若き女優アニャ・ロバーツ＝ジョイが怪演！
さらにアンのルームメイトであり、事件の真相を追うヨナ役には、『グランド・バザール』で観光客をターゲットに荒稼ぎをする詐欺師を熱演したトーマシン・マッケンナが演じる。

悪魔的デモニッシュな魅力で世界を虜にした『ヘルズ・ランウェイ』が今春、遂に日本にも解き放たれる─!
舞台はハリウッドスターや人気モデルが数多く暮らす、ロサンゼルス郊外の高級住宅地チェビオット・ヒルズ。
ここには全米から成功をめざす人々が集まり、熱気に満ちていた。
「私、もっと刺激的で胸のときめく人生を送りたいの」
漠然とそんな願望を抱いた若い田舎娘アンもその1人だった。

彼女は誰もが目を奪われる純粋な美しさに恵まれており、その美貌はすぐに人々の目に止まった。
一流プロデューサーとしてモデル業界の中心に君臨するジェフのもとで働き始めたアンは、瞬く間に人気モデルとなる。
仲のいいルームメイトのヨナとも一緒に仕事をする機会が増え、順風満帆な日々が続くと思われていた。

「アンはジェフと寝ている」突如、根も葉もない噂が広まりモデル業界に緊張が走る。
アンに対して、異常な嫉妬心を燃やすライバルたちが彼女を引きずりおろすために牙をむいたのだ。
負けず嫌いなアンもライバルたちと衝突し、苦しみながらも、着実に華やかなスターへの道を駆け上っていく。
次第にアン自身の中にも激しい野心が芽生え、究極の美のために手段を選ばなくなっていく。

一方でヨナは、アンを陥れようとする犯人探しをしていく中、衝撃の事実を知ることとなる。
運命の糸がほつれ、絡み合い、やがて狂い始めるふたりは誰もが想像し得ない境地にたどり着く。

アニャ・ロバーツ＝ジョイ as アン

トップモデルを夢見る16歳の女の子。
おっとりした見た目とは裏腹に勝気な性格で、モデル業界で頭角を現していく。

PROFILE
2006年、5人きょうだいの末っ子として米フロリダ州マイアミに生まれる。
幼少期をアルゼンチンとイギリスで過ごし、12歳で俳優を目指してニューヨークに渡る。
ストリート・モデルの創立者ベン・べイトマンに路上スカウトされモデル業をスタート。
2020年に俳優デビュー。翌年には主演映画『ポートフォリオ』で注目を浴び、ドラマ『ラブ&ロック』で大ブレイク。

トーマシン・マッケンナ as ヨナ

キューバからの移民で、アンのルームメイト。
大人しいが芯の強い女性で、都会の空気に慣れないアンをサポートする。

PROFILE
イスラエルの母とニュージーランドの父を持つ女優。
ミス・イスラエルのタイトルを獲得し、2010年度ミス・ユニバース世界大会に祖国を代表して参加した。
その後すぐに演技キャリアをスタートさせ、イスラエルのTVシリーズ『boycott』で主役を演じた。
2019年、初めてアメリカ映画に進出し、『フォクシー』でマイケル・フォスターを誘惑するハルを演じ注目を浴び、記憶に新しい『グランド・バザール』では老婆のメイクが一躍話題になった。

エマ・ロック as ソフィア

アンの同期で美しいブロンドヘアーの持ち主。
自信家でプライドが高く、自分より売れるアンを疎ましく思い嫌がらせをする。

PROFILE
2004年、アメリカ、ニューヨーク州生まれ。
幼い頃より習っていたバレエの道を目指したが、膝の怪我で断念。
その後、女優を目指してコロンビア大学で演劇を学ぶ。
マンハッタンのブレイクダンス集団「ポップ・ステップ・ブリゲイド」の舞台に立ち18年からは人気ダンス番組「ブレイキン・ナイト」に出演し、好評を博す。
『オーマイゴッドファーザー3』では本人役でカメオ出演し、その後『ダンス!ダンス!ダンス!』では主要キャストに抜擢される。
アニャとは2年前に公開した『エミリー、暖かい色』でも共演。
プライベートでも仲がいいことで知られている。

リリー・ジャスミン as エリザベス

スキャンダルで仕事を失った落ち目のモデル。
仕事が軌道に乗り出したころ、ぽっと出のアンに仕事を奪われ、対立を深めていく。

PROFILE
アメリカの俳優でプロデューサー。
国内外を問わず、テレビや映画、舞台で常に重要な役を演じている。
2018年、全6話のNetfilxオリジナル・ドラマ『ショコラティエ』では製作総指揮も務めた。
現在、ロマンチック・コメディー『スクール・オブ・アクシデント』に出演中で、アナ・デ・アリマス、ティモシー・シャメと共演している。

ジャッキー・チャオ as 美琳(メイリン)

女優業も兼ねる生真面目な中国人モデル。
急速に成長するアンに焦りと苛立ちを募らせる。

PROFILE
1996年、中国安徽省出身。
中国ではリンユー、チュンイェン、の2人と並び中国三大美女と呼ばれている。
TVドラマ『泡沫姫』で演じた役名「雪蘭」（シュェラン）の愛称でも親しまれる。
女優のほか歌手としても活動している。

サラ・デルヴィーニュ as ミア

ジェフの妻でカリスマモデル。
次第にアンの魅力に取りつかれていく夫をみて、心穏やかではない状態に。

PROFILE
1992年、ロンドン出身。
バーバリー、シャネルなどのキャンペーンやランウェイでモデルとして活躍。
インスタグラムのフォロワー数3,200万人という大人気のファッションアイコンとなる。
2010年にはマシュー・ホーン監督の『トラベラー』でジェイク・マンホールと共演し、映画デビュー。
KCコミックスのアクション大作『ボム・ボマー』ではスパイのイェレナを妖艶に演じた。

エノーラ・パーネル as ケイト

ミステリアスな雰囲気を醸し出すモデル。
薬物中毒者で周囲からは変人扱いされている。

PROFILE
ロサンゼルス生まれで、父は名カメラマンのロブ・パーネル、母は俳優のキャサリン・パーネルという芸能一家に育つ。
15歳のときTVのコメディ番組でブレイクし、『ハイスクール』でのスクリーンデビューを機に大学を辞め、俳優業に専念。
恋愛SFコメディ「タイムラブマシーン」で国際的な注目を集め、主人公を演じたシットコム「ガールズバンド」は6シーズン続く長寿ドラマとなった。
筋金入りのゲーマーとしても有名で、人気FPS『エージェント』ではアナウンス役の声を担当している。

マーゴット・ロビンソン as ナタリー

第一線で活躍するトップモデル。
モデル・オブ・ザ・イヤーにも輝いたことがあり、
アンがモデルを目指すきっかけとなった存在。

PROFILE
1985年、オーストラリア生まれ。
大学卒業後、女優の道を目指し、1999年にデビュー。
オーストラリアのTVドラマ「countryside（原題）」で国内人気を獲得した後、『ドライフラワー』で、米国進出を果たす。
その後『アリスタ ～スターへの道～』のアリスタ役で人気スターとなり、2017年には『コーラル・ウィンドウ』でアカデミー賞®主演女優賞にノミネートされ、実力派としても認められる。
近年の映画出演作に『バルーン』、『ファミリー・ミーティング』などがある。

ジョシュ・ハート as トミー

ハンサムな男性モデル。
プレイボーイで女性トラブルが絶えない。
短絡思考の男で、エリザベスに利用されてしまう。

PROFILE
ニュージーランドの俳優。
2021年に、ニュージーランドの大手制作会社サウス・パシフィック・ピクチャーズが手掛けるドラマ『ブロークンムーン』に起用され、ハワード役を演じた。

ディン・デハーン as ジェフ

アンの魅力を見出した敏腕プロデューサー。

PROFILE
オーストラリアの俳優、脚本、映画監督。
テレビや舞台で活躍を続け、国内外の20本以上の映画にも出演している。
実話を元にした犯罪サスペンス『セキュリティメンテナンス』に出演し、この作品は第70回カンヌ国際映画祭のある視点部門でプレミア上映された。
現在はステラ・マッカートニーの顔として同社の広告キャンペーンにも登場している。

エミリー・ブラウン

PROFILE
1988年頃から舞台に出演し始め、1996年、『ストレートフラッシュ』で映画デビューし、大ヒット映画『コールサイン』でハリウッド進出を果たしブレイク。
2000年頃からニューヨークのスクール・オブ・ビジュアル・アーツで映画製作を学び、2003年に『ツアーガイド』で長編映画監督デビュー。
『ぼくのかっこわるくて、いじわるで、大嫌いなおじ』で第85回アカデミー賞脚本賞を受賞し、『ガールズ・オブ・サマー』で、メディアから「この10年で最も素晴らしいラブ・ロマンス映画の1本」と高く評価される。
現在は危険な陰謀に巻き込まれたスタントマンを描いたリアルかつ斬新なアクション映画『Plummet （原題）』を撮影中。
